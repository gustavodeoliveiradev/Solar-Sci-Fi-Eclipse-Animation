# 🌌 Neon Eclipse Protocol

Projeto de estudo de animações avançadas em CSS, transformando um exercício simples em uma experiência visual Sci-Fi.

## 🚀 Demonstração
Acesse o projeto online: [(https://gustavodeoliveiradev.github.io/Solar-Sci-Fi-Eclipse-Animation/)]

## 🛠️ Tecnologias Utilizadas
- **HTML5**: Estrutura semântica.
- **CSS3**: Animações com `@keyframes`, pseudo-elementos (`::before`, `::after`) e efeitos de transparência.
- **Git/GitHub**: Controle de versão e hospedagem.

## 🧪 Experimentos Aplicados
Neste projeto, explorei conceitos além do básico:
1. **Estrelas Dinâmicas**: Uso de múltiplos `box-shadow` para criar um fundo espacial leve.
2. **Efeito Glitch**: Simulação de interferência de sinal usando animações não-lineares e `filter: blur`.
3. **Estética Sci-Fi**: Paleta de cores baseada em neon (Ciano e Magenta) para um visual futurista.
